# Covid_19_Classification_DataScience
This project was done as part of a Programming for Data Science unit coursework. The objective was to build and evaluate machine learning classification models capable of predicting whether an individual is likely to contract COVID-19 based on a structured dataset.

The project was completed collaboratively within a group setting, with contributions in data preprocessing, model implementation, evaluation, and comparative analysis.


## Objectives
Carry out Exploratory Data Analysis
Build a Random Forest Classifier and a Gradient Boosting Classifier
Evaluate and compare model performance
Analyze feature importance and draw practical insights from model outputs

##Technologies Used
Python
Libraries - Pandas, Numpy, Scikit-Learn, Matplotlib, Seaborn

##Methodology
1. Exploratory data Analysis
   * Understanding data patterns, relationships and anomalies
2.  Data Preprocessing
   * Handling missing values, Encoding categorical variables, Feature selection and preparation
3. Model Development and Evaluation
  * Implemented Random Forest Classifier and Gradient Boost Classifier
  * Used metrics such as Precision, Recall and F1-Score
  * Model tuning for improved performance
4. Feature Importance Analysis
  * Identified key features that are most useful for the models to make predictions
5. Discussion of practical implications of results.
  * This includes insights for future data quality checks and clinical focus
